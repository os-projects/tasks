Доступные команды:

    ls - вывести имена всех файлов в файловой системе

    cat> %filename% - писать в файл. Считываются все вводимые пользователем
                      в STDIN символы. Выход из этого режима осуществляется
                      по комбинации Ctrl+C.

    cat %filaneme% - вывести содержимое файла на экран

    rm %filename% - удалить файл

    touch %filename% - создать пустой файл, если такой ещё н существует

    getsize %filename%  - напечатать на экран размер файла в байтах

    exit - выйти из программы

    help - вывести это сообщение

    test %N% - запустить N потоков, где каждый бежит по файловой системе
               и выводит на экран размеры каждого файла. Для разграничения
               доступа к файлам используется Lamport's bakery algorithm, с
               одним файлом в один момент работает только один поток.

               *Вывод на экран при этом намеренно не синхронизируется, и
                потоки могут одновременно писать на экран с коллизиями, но
                только про разные файлы.
